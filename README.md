# GPA Dashboard — 성적 분석 대시보드

강남대학교 글로벌경영학부 경영학전공 / 데이터사이언스 복수전공 학업 성적을 시각화하는 웹 대시보드입니다.

## 미리보기

- **전체 평점**: 3.92 / 4.5 (총 139학점)
- **경영학 전공**: 3.94 (57학점)
- **DS 복수전공**: 4.28 (42학점)

## 주요 기능

### GPA 추이
2019-1학기부터 2025-2학기까지 학기별 평점 변화를 라인 차트로 표시합니다. 전체, 경영학, 데이터사이언스 세 트랙의 성적 추이를 한눈에 비교할 수 있습니다.

### DS 전공 분석
데이터사이언스 복수전공 14개 과목의 백분율 점수를 수평 막대 차트로 시각화합니다. 게이지 차트, 등급 분포(A+ 10과목, A 2과목, B+ 2과목), 주요 하이라이트를 함께 보여줍니다.

### 구성 분석
- **학기별 수강 학점 구성**: 교양, 경영학, DS 전공의 학점 비중 변화를 누적 막대 차트로 표시
- **전공별 성적 등급 비교**: 경영학(17과목)과 DS(14과목)의 등급 분포를 비교

## 실행 방법

별도의 빌드 과정 없이 `index.html` 파일을 브라우저에서 직접 열면 됩니다.

```bash
# 로컬에서 열기
open index.html

# 또는 간단한 HTTP 서버로 실행
python3 -m http.server 8000
```
